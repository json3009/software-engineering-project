Detailed Description

All story state information is included in the StoryState class, including global variables, read counts, the pointer to the current point in the story, the call stack (for tunnels, functions, etc), and a few other smaller bits and pieces.

You can save the current state using the json serialisation functions ToJson and LoadJson.

◆ ForceEnd()

void Ink.Runtime.StoryState.ForceEnd ( )

Ends the current ink flow, unwrapping the callstack but without affecting any variables.

Useful if the ink is (say) in the middle a nested tunnel, and you want it to reset so that you can divert elsewhere using ChoosePathString(). Otherwise, after finishing the content you diverted to, it would continue where it left off. Calling this is equivalent to calling -> END in ink.
