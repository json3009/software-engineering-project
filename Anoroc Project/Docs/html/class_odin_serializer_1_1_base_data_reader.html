Detailed Description

Provides basic functionality and overridable abstract methods for implementing a data reader.

If you inherit this class, it is VERY IMPORTANT that you implement each abstract method to the exact specifications the documentation specifies.

See also
BaseDataReaderWriter, IDataReader

◆ ExitArray()

abstract bool OdinSerializer.BaseDataReader.ExitArray ( )
pure virtual

Exits the closest array.

This method will keep skipping entries using IDataReader.SkipEntry(DeserializationContext) until an EntryType.EndOfArray is reached, or the end of the stream is reached.

This call MUST have been preceded by a corresponding call to IDataReader.EnterArray(out long).

This call will change the values of the IDataReader.IsInArrayNode, IDataReader.CurrentNodeName, IDataReader.CurrentNodeId and IDataReader.CurrentNodeDepth to the correct values for the node that was prior to the exited array node.

Returns
true if the method exited an array, false if it reached the end of the stream.

Implements OdinSerializer.IDataReader.

Implemented in OdinSerializer.SerializationNodeDataReader, OdinSerializer.JsonDataReader, and OdinSerializer.BinaryDataReader.

◆ SkipEntry()

virtual void OdinSerializer.BaseDataReader.SkipEntry ( )
virtual

Skips the next entry value, unless it is an EntryType.EndOfNode or an EntryType.EndOfArray.

If the next entry value is an EntryType.StartOfNode or an EntryType.StartOfArray, all of its contents will be processed, deserialized and registered in the deserialization context, so that internal reference values are not lost to entries further down the stream.

Implements OdinSerializer.IDataReader.
