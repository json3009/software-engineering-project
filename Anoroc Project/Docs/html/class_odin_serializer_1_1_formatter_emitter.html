Detailed Description

Utility class for emitting formatters using the System.Reflection.Emit namespace.

NOTE: Some platforms do not support emitting. Check whether you can emit on the current platform using EmitUtilities.CanEmit.

Gets an emitted formatter for a given type.

NOTE: Some platforms do not support emitting. On such platforms, this method logs an error and returns null. Check whether you can emit on the current platform using EmitUtilities.CanEmit.

Parameters
typeThe type to emit a formatter for.
policyThe serialization policy to use to determine which members the emitted formatter should serialize. If null, SerializationPolicies.Strict is used.
Returns
The type of the emitted formatter.
Exceptions
System.ArgumentNullExceptionThe type argument is null.
