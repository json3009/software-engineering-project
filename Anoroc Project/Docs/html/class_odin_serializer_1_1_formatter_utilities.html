Creates a fake Unity null value of a given type, for the given UnityEngine.Object-derived owning type.

Unity uses these kinds of values to indicate missing object references.

Parameters

owningTypeType of the owning value. This is the value which changes the MissingReferenceException which you get.
Returns
A fake Unity null value of a given type.
Exceptions

System.ArgumentExceptionThe type given in the nullType parameter is not a Unity object. or The type given in the owningType parameter is not a Unity object.

◆ IsPrimitiveArrayType()

static bool OdinSerializer.FormatterUtilities.IsPrimitiveArrayType ( Type  type)
static

Determines whether a given type is a primitive array type.

Namely, arrays with primitive array types as elements are primitive arrays.

The following types are primitive array types: char, sbyte, short, int, long, byte, ushort, uint, ulong, decimal, bool, float, double and Guid.

Parameters
typeThe type to check.
Returns
true if the given type is a primitive array type; otherwise, false.
