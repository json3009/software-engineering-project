◆ ExitArray()

override bool OdinSerializer.JsonDataReader.ExitArray ( )
virtual

Exits the closest array.

This method will keep skipping entries using IDataReader.SkipEntry(DeserializationContext) until an EntryType.EndOfArray is reached, or the end of the stream is reached.

This call MUST have been preceded by a corresponding call to IDataReader.EnterArray(out long).

This call will change the values of the IDataReader.IsInArrayNode, IDataReader.CurrentNodeName, IDataReader.CurrentNodeId and IDataReader.CurrentNodeDepth to the correct values for the node that was prior to the exited array node.

Returns
true if the method exited an array, false if it reached the end of the stream.
