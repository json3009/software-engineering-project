Detailed Description

Minimal baseline formatter.

Doesn't come with all the bells and whistles of any of the other BaseFormatter classes. Common serialization conventions aren't automatically supported, and common deserialization callbacks are not automatically invoked.

Template Parameters
TThe type which can be serialized and deserialized by the formatter.
