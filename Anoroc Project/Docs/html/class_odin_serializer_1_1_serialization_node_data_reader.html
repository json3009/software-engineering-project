◆ EnterArray()

override bool OdinSerializer.SerializationNodeDataReader.EnterArray ( out long  length)
virtual

Tries to enters an array node.

This will succeed if the next entry is an EntryType.StartOfArray.

This call MUST (eventually) be followed by a corresponding call to IDataReader.ExitArray(DeserializationContext)

This call will change the values of the IDataReader.IsInArrayNode, IDataReader.CurrentNodeName, IDataReader.CurrentNodeId and IDataReader.CurrentNodeDepth properties to the correct values for the current array node.

Parameters
lengthThe length of the array that was entered.
Returns
true if an array was entered, otherwise false

◆ ReadPrimitiveArray< T >()

override bool OdinSerializer.SerializationNodeDataReader.ReadPrimitiveArray< T > ( out T[]  array)
virtual

Reads a primitive array value.

This call will succeed if the next entry is an EntryType.PrimitiveArray.

If the call fails (and returns false), it will skip the current entry value, unless that entry is an EntryType.EndOfNode or an EntryType.EndOfArray.

Template Parameters
TThe element type of the primitive array. Valid element types can be determined using FormatterUtilities.IsPrimitiveArrayType(Type).
Parameters
arrayThe resulting primitive array.
Returns
true if reading a primitive array succeeded, otherwise false
Exceptions
System.ArgumentExceptionType + typeof(T).Name + is not a valid primitive array type.
