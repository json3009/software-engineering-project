int PlayFab.ClientModels.VirtualCurrencyRechargeTime.RechargeMax

Maximum value to which the regenerating currency will automatically increment.

Note that it can exceed this value through use of the AddUserVirtualCurrency API call. However, it will not regenerate automatically until it has fallen below this value.
