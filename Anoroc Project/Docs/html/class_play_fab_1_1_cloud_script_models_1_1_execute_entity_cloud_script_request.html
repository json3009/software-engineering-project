◆ RevisionSelection

CloudScriptRevisionOption? PlayFab.CloudScriptModels.ExecuteEntityCloudScriptRequest.RevisionSelection

Option for which revision of the CloudScript to execute.

'Latest' executes the most recently created revision, 'Live' executes the current live, published revision, and 'Specific' executes the specified revision. The default value is 'Specific', if the SpecificRevision parameter is specified, otherwise it is 'Live'.
