◆ GsdkAlias

string PlayFab.MultiplayerModels.GameCertificateReference.GsdkAlias

An alias for the game certificate.

The game server will reference this alias via GSDK config to retrieve the game certificate. This alias is used as an identifier in game server code to allow a new certificate with different Name field to be uploaded without the need to change any game server code to reference the new Name.
