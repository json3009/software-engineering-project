Detailed Description

The Groups API is designed for any permanent or semi-permanent collections of Entities (players, or non-players).

If you want to make Guilds/Clans/Corporations/etc., then you should use groups. Groups can also be used to make chatrooms, parties, or any other persistent collection of entities.
