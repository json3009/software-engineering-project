Detailed Description

Provides a set of methods for reading data stored in a format written by a corresponding IDataWriter class.

If you implement this interface, it is VERY IMPORTANT that you implement each method to the exact specifications the documentation specifies.

It is strongly recommended to inherit from the BaseDataReader class if you wish to implement a new data reader.

See also
System.IDisposable

◆ EnterArray()

bool OdinSerializer.IDataReader.EnterArray ( out long  length)

Tries to enters an array node.

This will succeed if the next entry is an EntryType.StartOfArray.

This call MUST (eventually) be followed by a corresponding call to IDataReader.ExitArray(DeserializationContext)

This call will change the values of the IDataReader.IsInArrayNode, IDataReader.CurrentNodeName, IDataReader.CurrentNodeId and IDataReader.CurrentNodeDepth properties to the correct values for the current array node.

Parameters

Returns

◆ ExitArray()

bool OdinSerializer.IDataReader.ExitArray ( )

Exits the closest array.

This method will keep skipping entries using IDataReader.SkipEntry(DeserializationContext) until an EntryType.EndOfArray is reached, or the end of the stream is reached.

This call MUST have been preceded by a corresponding call to IDataReader.EnterArray(out long).

This call will change the values of the IDataReader.IsInArrayNode, IDataReader.CurrentNodeName, IDataReader.CurrentNodeId and IDataReader.CurrentNodeDepth to the correct values for the node that was prior to the exited array node.

Returns
true if the method exited an array, false if it reached the end of the stream.

Implemented in OdinSerializer.SerializationNodeDataReader, OdinSerializer.JsonDataReader, OdinSerializer.BinaryDataReader, and OdinSerializer.BaseDataReader.

◆ SkipEntry()

void OdinSerializer.IDataReader.SkipEntry ( )

Skips the next entry value, unless it is an EntryType.EndOfNode or an EntryType.EndOfArray.

If the next entry value is an EntryType.StartOfNode or an EntryType.StartOfArray, all of its contents will be processed, deserialized and registered in the deserialization context, so that internal reference values are not lost to entries further down the stream.

Implemented in OdinSerializer.BaseDataReader.
