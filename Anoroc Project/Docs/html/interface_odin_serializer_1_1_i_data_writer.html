Detailed Description

Provides a set of methods for reading data stored in a format that can be read by a corresponding IDataReader class.

If you implement this interface, it is VERY IMPORTANT that you implement each method to the exact specifications the documentation specifies.

It is strongly recommended to inherit from the BaseDataWriter class if you wish to implement a new data writer.

See also
System.IDisposable
