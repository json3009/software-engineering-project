Detailed Description

Serializes and deserializes a given type.

Serializes and deserializes a given type T.

NOTE that if you are implementing a custom formatter and registering it using the CustomFormatterAttribute, it is not enough to implement IFormatter - you have to implement IFormatter<T>.

Template Parameters
TThe type which can be serialized and deserialized by the formatter.
