Detailed Description

Internal structure used to point to a particular / current point in the story.

Where Path is a set of components that make content fully addressable, this is a reference to the current container, and the index of the current piece of content within that container. This scheme makes it as fast and efficient as possible to increment the pointer (move the story forwards) in a way that's as native to the internal engine as possible.
